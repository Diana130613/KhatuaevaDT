# Тема 10: Графы

## Цель работы: 
Изучить основные понятия теории графов и алгоритмы работы с ними. Освоить представления графов в памяти и основные алгоритмы обхода. Получить практические навыки реализации алгоритмов на графах и анализа их сложности.

## Задание:
1. Реализовать различные представления графов (матрица смежности, список смежности).
2. Реализовать алгоритмы обхода графов (BFS, DFS).
3. Реализовать алгоритмы поиска кратчайших путей и компонент связности.
4. Провести сравнительный анализ эффективности разных представлений графов.
5. Решить практические задачи на графах.

## Шаги выполнения:
1. Создание проекта: 
- Создать файлы graph_representation.py, graph_traversal.py, shortest_path.py.

2. Реализация представлений графов:
- Класс для матрицы смежности
- Класс для списка смежности
- Методы добавления/удаления вершин и рёбер
- Для каждого представления указать потребление памяти и сложность операций.

3. Реализация алгоритмов обхода:
- BFS с возвратом расстояний и путей
- DFS (рекурсивный и итеративный)
- Указать сложность и особенности каждого алгоритма.

4. Реализация алгоритмов на графах:
- Поиск компонент связности
- Топологическая сортировка
- Алгоритм Дейкстры для взвешенных графов

5. Тестирование:
- Написать unit-тесты для проверки корректности работы всех алгоритмов
- Протестировать на графах разных типов и размеров

6. Экспериментальное исследование:
- Сравнить время выполнения операций для разных представлений графов
- Исследовать масштабируемость алгоритмов на больших графах  
ВАЖНО: Все замеры проводить на одной вычислительной машине

7. Визуализация:
- Построить графики зависимости времени операций от размера графа
- Визуализировать графы и результаты работы алгоритмов

8. Решение задач:
- Найти кратчайший путь в лабиринте
- Определить связность сети
- Решить задачу на топологическую сортировку

9. Анализ результатов:  
- Сравнить эффективность разных представлений графов
- Проанализировать применимость алгоритмов для разных типов задач

10. Оформление отчета: 
Результаты оформить в файле README.md. Код должен соответствовать
PEP8.
