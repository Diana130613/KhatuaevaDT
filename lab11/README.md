# Тема 11: Алгоритмы на строках

## Цель работы: 
Изучить специализированные алгоритмы для эффективной работы со строками. Освоить
методы поиска подстрок, вычисление префикс-функции и Z-функции. Получить практические навыки
реализации и анализа алгоритмов обработки строк, исследовать их производительность.

## Задание:
1. Реализовать вычисление префикс-функции для строки.
2. Реализовать алгоритм Кнута-Морриса-Пратта для поиска подстроки.
3. Реализовать вычисление Z-функции.
4. Реализовать один из дополнительных алгоритмов поиска подстроки.
5. Провести сравнительный анализ эффективности алгоритмов на различных данных.

## Шаги выполнения:
1. Создание проекта: Создать файлы prefix_function.py, kmp_search.py, z_function.py, string_matching.py.

2. Реализация базовых алгоритмов:
- Вычисление префикс-функции
- Алгоритм KMP с использованием префикс-функции
- Вычисление Z-функции
- Поиск подстроки с использованием Z-функции
- Для каждого алгоритма указать временную и пространственную сложность.

3. Реализация дополнительного алгоритма:
- Алгоритм Рабина-Карпа

4. Тестирование:
- Написать unit-тесты для проверки корректности работы всех алгоритмов
- Протестировать на строках разных типов (случайные, периодические, с повторениями)

5. Экспериментальное исследование:
- Сравнить время выполнения алгоритмов на различных паттернах
- Исследовать поведение алгоритмов в худшем и лучшем случаях
- ВАЖНО: Все замеры проводить на одной вычислительной машине

6. Визуализация:
- Построить графики зависимости времени выполнения от длины текста и паттерна
- Визуализировать работу префикс-функции и Z-функции

7. Решение задач:
- Найти все вхождения паттерна в тексте
- Решить задачу на поиск периода строки
- Реализовать проверку циклического сдвига строк

8. Анализ результатов:
- Сравнить эффективность разных алгоритмов поиска подстрок
- Проанализировать влияние характеристик строк на производительность
- Определить оптимальные области применения каждого алгоритма

9. Оформление отчета: 
Результаты оформить в файле README.md. Код должен соответствовать
PEP8.
